# 飞书登录流程（H5 网页应用）

## 概述

用户从飞书工作台或消息卡片点击进入 Apex H5 应用，飞书负责鉴权，系统无独立登录页。前端通过飞书 JSSDK 获取临时授权码（code），后端用 code 换取用户信息并签发 JWT，完成免密登录。

---

## 前置条件（飞书开放平台配置）

| 配置项 | 值 |
|--------|-----|
| 应用类型 | 企业自建应用 |
| 功能 | 网页应用（H5） |
| 网页 URL（入口） | `https://your-domain.com` |
| 可信域名 | `https://your-domain.com` |
| 所需权限 | `contact:user.base:readonly` |

> 开发阶段可用 cloudflared 临时域名替代，生产环境替换为正式域名后重新配置即可。

## 完整登录流程

```
┌──────────────────────────────────────────────────────────────────┐
│                      用户侧（飞书客户端）                           │
└──────────────────────────────────────────────────────────────────┘
          │
          │  1. 点击工作台应用图标 或 消息卡片按钮
          ▼
┌──────────────────────────────────────────────────────────────────┐
│                      飞书平台                                      │
│  打开 H5 入口 URL（无 code 参数）                                   │
│  https://your-domain.com/                                        │
└──────────────────────────────────────────────────────────────────┘
          │
          │  2. WebView 加载 H5 页面
          ▼
┌──────────────────────────────────────────────────────────────────┐
│                      前端 H5（React）                              │
│                                                                  │
│  App 启动时执行 useAuth() Hook：                                   │
│                                                                  │
│  ┌──────────────────┐      ┌─────────────────────────────────┐  │
│  │ localStorage 有   │      │ localStorage 无 token            │  │
│  │ 有效 JWT token    │      │ 或 token 已过期                   │  │
│  └────────┬─────────┘      └────────────┬────────────────────┘  │
│           │                             │                        │
│           ▼                             ▼                        │
│     直接进入 Dashboard         调用飞书 JSSDK                      │
│                                tt.requestAuthCode({             │
│                                  appId: 'cli_xxx'               │
│                                })                               │
│                                → 获得临时 code（约 5 分钟有效）    │
└──────────────────────────────────────────────────────────────────┘
          │
          │  3. 前端携带 code 请求后端
          │  POST /api/auth/feishu   { code }
          ▼
┌──────────────────────────────────────────────────────────────────┐
│                      后端（FastAPI）                               │
│                                                                  │
│  Step 1  用 code 换 user_access_token                             │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │ POST https://open.feishu.cn/open-apis/authen/v1/         │    │
│  │      access_token                                        │    │
│  │ Body: { grant_type, client_id, client_secret, code }    │    │
│  │ → 返回 user_access_token                                 │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
│  Step 2  用 user_access_token 获取用户信息                         │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │ GET https://open.feishu.cn/open-apis/authen/v1/user_info│    │
│  │ Header: Authorization: Bearer <user_access_token>       │    │
│  │ → 返回 open_id / name / avatar_url / email              │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
│  Step 3  写入 / 更新 users 表                                      │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │ upsert by open_id                                        │    │
│  │ 字段：open_id, name, avatar_url, email, last_login_at   │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                  │
│  Step 4  签发 JWT                                                 │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │ payload: { user_id, open_id, name, exp: now + 7d }      │    │
│  │ 返回给前端                                                │    │
│  └─────────────────────────────────────────────────────────┘    │
└──────────────────────────────────────────────────────────────────┘
          │
          │  4. 返回 JWT
          ▼
┌──────────────────────────────────────────────────────────────────┐
│                      前端 H5（React）                              │
│                                                                  │
│  localStorage.setItem('token', jwt)                              │
│  清除 URL 中残留参数（history.replaceState）                        │
│  跳转进入 Dashboard                                               │
└──────────────────────────────────────────────────────────────────┘
          │
          │  5. 后续所有 API 请求
          ▼
┌──────────────────────────────────────────────────────────────────┐
│  请求头统一携带 Authorization: Bearer <JWT>                        │
│  后端 middleware 验证 JWT → 注入 current_user → 返回数据            │
└──────────────────────────────────────────────────────────────────┘
```

## Token 失效处理

```
前端请求收到 401 响应
        │
        ▼
清除 localStorage token
        │
        ▼
重新调用 tt.requestAuthCode() 获取新 code
（用户无感知，自动静默刷新）
        │
        ▼
若 requestAuthCode 也失败（用户离开飞书环境）
        │
        ▼
提示"请在飞书中打开本应用"
```

## 关键代码结构

### 前端 — JSSDK 引入

```html
<!-- index.html -->
<script src="https://lf1-cdn-tos.bytegoofy.com/goofy/lark/op/h5-js-sdk-1.5.43.js"></script>
```

### 前端 — useAuth Hook 骨架

```typescript
// src/hooks/useAuth.ts
import { useEffect } from 'react'

declare const tt: any

export function useAuth() {
  useEffect(() => {
    const token = localStorage.getItem('token')
    if (token && !isExpired(token)) return

    // 调用飞书 JSSDK 获取 code
    window.h5sdk.ready(() => {
      tt.requestAuthCode({
        appId: import.meta.env.VITE_FEISHU_APP_ID,
        success: async (res: { code: string }) => {
          const { jwt } = await fetch('/api/auth/feishu', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ code: res.code })
          }).then(r => r.json())
          localStorage.setItem('token', jwt)
          window.location.replace('/')
        },
        fail: (err: unknown) => {
          console.error('飞书授权失败', err)
        }
      })
    })
  }, [])
}
```

### 后端 — 登录接口骨架

```python
# api/routers/auth.py
from fastapi import APIRouter
import httpx, jwt, os
from datetime import datetime, timedelta

router = APIRouter()

FEISHU_APP_ID = os.getenv("FEISHU_APP_ID")
FEISHU_APP_SECRET = os.getenv("FEISHU_APP_SECRET")
JWT_SECRET = os.getenv("JWT_SECRET")

@router.post("/api/auth/feishu")
async def feishu_login(body: dict):
    code = body["code"]

    # Step 1: code → user_access_token
    async with httpx.AsyncClient() as client:
        token_res = await client.post(
            "https://open.feishu.cn/open-apis/authen/v1/access_token",
            json={
                "grant_type": "authorization_code",
                "client_id": FEISHU_APP_ID,
                "client_secret": FEISHU_APP_SECRET,
                "code": code
            }
        )
        user_access_token = token_res.json()["data"]["access_token"]

        # Step 2: 获取用户信息
        user_res = await client.get(
            "https://open.feishu.cn/open-apis/authen/v1/user_info",
            headers={"Authorization": f"Bearer {user_access_token}"}
        )
        user = user_res.json()["data"]

    # Step 3: upsert users 表（略，接入 DB 后补充）

    # Step 4: 签发 JWT
    payload = {
        "open_id": user["open_id"],
        "name": user["name"],
        "exp": datetime.utcnow() + timedelta(days=7)
    }
    token = jwt.encode(payload, JWT_SECRET, algorithm="HS256")

    return {"jwt": token}
```

## 注意事项

- `requestAuthCode` 获取的 code **一次性使用**，5 分钟内有效，用完即失效
- JWT 建议有效期 7 天，到期前前端静默刷新
- 飞书 JSSDK 只在飞书 WebView 环境下有效，本地浏览器调试需 mock
- 生产环境必须 HTTPS，可信域名需与实际部署域名完全一致
